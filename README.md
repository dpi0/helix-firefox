## Features

- **Copy URL**: Copy the current page's URL with `Alt + M`.
- **Switch Tabs**: Switch to the previously active tab with `Alt + A`.
- **Remove Playlist**: Clean up YouTube URLs and open the video in a new tab with `Ctrl + Shift + Y`.
